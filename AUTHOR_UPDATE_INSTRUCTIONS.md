## Photo Requirements
- **Filename:** `author-photo.jpg`
- **Upload to:** `images/` folder in your GitHub repo
- **Recommended size:** 400×400 pixels (square works best for circular crop)
- **Format:** JPG or PNG
- **Style:** Will be displayed as a circle with white border and shadow

## How to Update Your Live Site

### Step 1: Upload Your Photo (2 minutes)
1. Go to your GitHub repo: `https://github.com/YOUR_USERNAME/thedigitalspecies`
2. Navigate to the `images/` folder
3. Click "Add file" → "Upload files"
4. Upload your `author-photo.jpg` file
5. Commit with message: "Add author photo"

### Step 2: Update the HTML (3 minutes)
1. In your GitHub repo, click on `index.html`
2. Click the pencil icon (Edit)
3. Find the "About Author" section (search for `<section class="author-section">`)
4. Replace the entire section with the new version from the updated package
5. Or copy/paste from: `/Users/sydbot/.openclaw/workspace/thedigitalspecies-launch/index.html` (lines 196-224)
6. Commit with message: "Update author bio"

### Step 3: Update the CSS (2 minutes)
1. In your GitHub repo, click `css/style.css`
2. Click the pencil icon (Edit)
3. Find the `.author-section` styles (around line 1001)
4. Replace with the updated CSS from the new package
5. Or copy/paste from: `/Users/sydbot/.openclaw/workspace/thedigitalspecies-launch/css/style.css` (lines 1001-1095)
6. Commit with message: "Update author section styles"

### Wait 1-2 minutes for GitHub Pages to rebuild, then check your site!
